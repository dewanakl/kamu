## About Kamu

Kamu adalah framework yang sangat sederhana. Saking sederhananya, dokumentasinya juga apa adanya :)

> Intinya mirip dengan laravel, karena terinspirasi darinya

## How to run ?
- `clone or download this repository`
- `cp .env.example .env`
- `php saya key`
- `php saya coba`
